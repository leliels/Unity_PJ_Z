# 02 - 游戏设计文档

## 1. 游戏概述

一款类似《快乐爱消除》的 2D 休闲方块消除游戏。玩家在固定大小的棋盘上放置不同形状的方块，填满整行或整列即可消除得分。通过关卡模式一关一关推进，每关有特定的目标和步数限制。

## 2. 核心玩法规则

### 2.1 棋盘

- 固定大小的方格棋盘（**待确认：8×8 还是 10×10？建议先做 8×8**）
- 每个格子有两种状态：空 / 已占用
- 棋盘初始状态为全空（或根据关卡配置有预置方块）

### 2.2 方块

- 屏幕下方的**候选区**同时显示 **3 个**方块组合供玩家选择
- 方块组合由若干小格子组成，形状各异（类似俄罗斯方块的形状）
- 玩家用手指/鼠标**拖拽**方块到棋盘上的空位放置
- 方块只能放在完全空的位置上，不能重叠
- 放置后不能移动、不能旋转（**待确认：是否允许旋转？**）
- 3 个候选方块用完后，自动刷新新的 3 个

### 2.3 方块形状表（初步设定）

> 以下为基础形状，后续可扩充。

| 形状名称 | 示意 | 占格数 |
|---------|------|--------|
| 单格 | ■ | 1 |
| 横二 | ■■ | 2 |
| 竖二 | ■<br>■ | 2 |
| 横三 | ■■■ | 3 |
| 竖三 | ■<br>■<br>■ | 3 |
| L 形 | ■<br>■■ | 3 |
| 田字 | ■■<br>■■ | 4 |
| T 形 | ■■■<br>&nbsp;■ | 4 |
| 横四 | ■■■■ | 4 |
| 横五 | ■■■■■ | 5 |

> **待补充**：完整的方块形状列表和各形状的出现概率/权重。

### 2.4 消除机制

- **行消除**：某一行的所有格子都被填满 → 该行所有方块被消除
- **列消除**：某一列的所有格子都被填满 → 该列所有方块被消除
- **同时消除**：如果一次放置同时填满了多行/多列，全部同时消除（连消奖励）
- 消除后，上方的方块**不会**下落（区别于传统三消游戏）

### 2.5 计分规则（初步设定）

| 事件 | 得分 |
|------|------|
| 放置方块 | 每格 1 分 |
| 消除 1 行/列 | 10 分 |
| 同时消除 2 行/列 | 30 分 |
| 同时消除 3 行/列 | 60 分 |
| 同时消除 4+ 行/列 | 100 分 |

> **待补充**：连击加分、Combo 系统等更详细的计分规则。

## 3. 关卡系统

### 3.1 关卡基本要素

每个关卡包含以下配置参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| 关卡编号 | 唯一标识 | Level_001 |
| 目标分数 | 达到此分数即过关 | 500 |
| 步数限制 | 可放置方块的总次数 | 20 |
| 棋盘大小 | 棋盘的行×列 | 8×8 |
| 预置方块 | 开局时已有的方块 | 待补充 |
| 可用方块池 | 本关可出现的方块形状 | 待补充 |
| 星级评价 | 1星/2星/3星的分数线 | 待补充 |

### 3.2 关卡流程

```
进入关卡 → 显示关卡目标 → 开始游戏
  ↓
每一步：选择候选方块 → 拖拽放置 → 检查消除 → 更新分数 → 剩余步数-1
  ↓
判定结果：
  ├─ 分数 ≥ 目标分数 → 过关！显示星级评价
  └─ 步数用尽且分数不足 → 失败，可选重试或返回
```

### 3.3 关卡进度存档

- 记录玩家已通过的最高关卡
- 记录每关的最高分和星级
- **存储方式**：使用 Unity 的 `PlayerPrefs`（简单方案，够用即可）

     ↓

     ↓

     ↓

     ↓

│          │←────│         │

     ↓
