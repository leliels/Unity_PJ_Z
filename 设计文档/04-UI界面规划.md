# 04 - UI 界面规划

> 本文档列出游戏中所有需要的 UI 界面，以及每个界面包含的元素。  
> 美术同学可以根据这个列表来设计 UI 原型和切图。

## 1. 界面总览

| 界面 | 所属场景 | 说明 |
|------|---------|------|
| 主菜单 | MainMenu | 游戏启动后的首页 |
| 关卡选择 | MainMenu | 展示所有关卡 |
| 关卡目标弹窗 | Game | 进入关卡前显示本关目标 |
| 游戏主界面（HUD） | Game | 游戏过程中的信息展示 |
| 暂停弹窗 | Game | 暂停游戏 |
| 过关结算弹窗 | Game | 显示过关成绩 |
| 失败结算弹窗 | Game | 显示失败信息 |
| 设置弹窗 | 通用 | 音量等设置 |

## 3. UI 技术说明

- 使用 Unity **uGUI** (Canvas + EventSystem)
- 所有弹窗使用同一个 Canvas，通过 **显示/隐藏 Panel** 切换
- Canvas 设置为 **Screen Space - Overlay** 模式
- Canvas Scaler 使用 **Scale With Screen Size**，参考分辨率 **1080 × 1920**（竖屏）
- **待确认：游戏是横屏还是竖屏？建议竖屏，更适合手机操作**

## 4. 美术同学需要提供的 UI 素材

| 素材类型 | 数量（估） | 说明 |
|---------|-----------|------|
| 按钮图片 | 2~3 种样式 | 主按钮、次按钮、小按钮 |
| 弹窗背景 | 1~2 种 | 圆角矩形面板 |
| 图标 | 5~8 个 | 齿轮、返回箭头、星星、锁、音乐、音效等 |
| 数字字体 | 1 套 | 分数显示用（可用免费字体） |
| 进度条 | 1 套 | 分数进度条底图 + 填充图 |
