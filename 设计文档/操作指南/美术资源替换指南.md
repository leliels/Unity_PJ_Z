## 一、当前资源架构

游戏通过 `Resources.Load<Sprite>()` 从 `Assets/Resources/Art/` 目录加载美术资源。  
如果资源不存在，会自动 fallback 到代码生成的纯色方块（不会崩溃）。

### 目录结构

```
Assets/Resources/Art/
├── Blocks/              # 方块素材
│   └── blk_base.png     # ← 方块基础形状（当前用 FK.png 临时替代）
│
├── Board/               # 棋盘素材
│   ├── brd_cell.png     # ← 棋盘空格（当前用 FK.png 临时替代）
│   ├── brd_frame.png    # ← 棋盘边框底板
│   └── brd_cell_dark.png # ← 棋盘深色格子（备用）
│
├── Backgrounds/         # 背景
│   └── bg_game.png      # ← 游戏背景（已接入，来自「背景.png」）
│
└── UI/                  # UI 素材（M3 阶段接入）
    ├── Buttons/
    │   ├── btn_back.png          # 返回按钮（蓝色箭头）
    │   ├── btn_primary.png       # 主按钮（绿色长条）

    │   ├── ico_music.png         # 音乐图标

    │   ├── ico_vibration.png     # 震动图标

    └── Panels/
        ├── pnl_score_bg.png      # 分数栏底板
```

## 三、如何替换正式美术资源

### 方法 1：直接替换 PNG 文件

1. 将正式美术 PNG 文件**直接替换** `Assets/Resources/Art/` 下对应的文件
2. 保持**文件名完全一致**（如 `blk_base.png`、`brd_cell.png`、`bg_game.png`）
3. 回到 Unity 编辑器，等待自动刷新（或手动 `Ctrl+R`）
4. 运行游戏查看效果

### 方法 2：添加 6 色方块（推荐！大幅提升视觉效果）

如果美术能提供 6 色独立方块切图：

1. 将 6 个 PNG 放到 `Assets/Resources/Art/Blocks/` 目录下
2. 命名为：`blk_color1.png` ~ `blk_color6.png`（对应红、橙、黄、绿、蓝、紫）
3. 代码中需要修改 `SpriteUtils.cs` 和 `BlockSpawner.cs`，将单色着色改为按颜色加载不同 Sprite（这一步由 AI 完成）

### Unity Import Settings 要求

替换/新增 PNG 后，确保在 Unity Inspector 中设置：

| 设置项 | 值 |
|-------|-----|
| Texture Type | **Sprite (2D and UI)** |
| Sprite Mode | Single |
| Pixels Per Unit | **128**（推荐，适配 0.9 的 CellSize） |
| Filter Mode | Bilinear |
| Compression | None 或 Low Quality |

## 四、代码调用关系图

```
SceneBootstrap.Awake()
├── SetupCamera()           → 配置相机
├── CreateBackground()      → 加载 bg_game.png 作为全屏背景
├── CreateManagers()        → 创建各 Manager
│   ├── BoardManager.Init()
│   │   └── CreateBoardVisuals()  → SpriteUtils.CellSprite (brd_cell.png)
│   └── BlockSpawner.Init()
│       └── CreateBlockVisual()   → SpriteUtils.BlockSprite (blk_base.png) + Color 着色
└── CreateUI()              → 代码创建 UI（M3 改为 Prefab 后使用 UI 素材）
```

## 五、当前视觉效果说明

- **背景**：✅ 已使用正式美术背景（棕色菱形纹理 + 棋盘轮廓）
- **棋盘格子**：⚠️ 使用 FK.png（灰色圆角方块）着色为深色，外观比纯色方块好很多，但与效果图的凹陷格子稍有差异
- **方块**：⚠️ 使用 FK.png（灰色圆角方块）着色为 6 种颜色。外观有圆角光泽感，但没有效果图中的渐变和高光细节
- **UI**：❌ 仍为代码生成的纯色 UI，M3 阶段替换

> **结论**：当前换皮后已达到"能见人"的最低标准。如果美术能提供 6 色独立方块切图，视觉效果会大幅提升。
