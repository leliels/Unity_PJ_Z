# 01 - 项目总览

## 1. 项目基本信息

| 项目 | 内容 |
|------|------|
| **项目暂定名** | 快乐消消乐（待定） |
| **游戏类型** | 2D 休闲益智 · 方块消除 |
| **参考游戏** | 《快乐爱消除》（App Store / TapTap 可搜到） |
| **项目目标** | 做一个最小化可玩的完整消除游戏，跑通从开发到上线的全流程 |
| **目标平台** | 先做 PC（Standalone），后续考虑移动端 |

## 2. 团队成员与分工

| 角色 | 主要职责 |
|------|---------|
| **技术美术（你）** | 项目统筹、驱动 AI 编写代码、整合资源、构建项目 |
| **美术** | 游戏内所有视觉素材：方块、棋盘、背景、UI 图标等 |
| **特效** | 消除特效、得分飘字、过关动画等视觉效果 |

> 注：程序部分主要由 AI 辅助完成，技术美术负责驱动和验收。

## 4. 项目范围定义

### 第一版（MVP）要做的

- [x] 基础棋盘 + 方块放置 ✅ M1 已完成
- [x] 行/列填满消除 ✅ M1 已完成
- [x] 基础 UI ✅ M1 已完成（分数 HUD + GameOver 面板）
- [ ] 新计分数值系统（16^N 消除分 + Combo）— M2 进行中
- [ ] 美术资源替换（换皮）— M2 进行中
- [ ] 关卡系统（限定步数 + 目标分数）— M3 待开发

- 广告 / 内购（SDK 已装，但后期再接入）
- 剧情 / 角色系统

## 5. 项目目录现状

```
Unity_PJ_Z/
├── Assets/
│   ├── Art/                    # 美术资源（效果图 + UI 切图）
│   │   ├── 拆分资源/           # 美术提供的切图（按钮、数字、图标、背景）
│   │   └── 拆分资源/效果图/    # 参考效果图
│   ├── Resources/              # 运行时资源
│   ├── Scenes/                 # 场景文件
│   │   └── Boot.unity          # 游戏主场景（挂 SceneBootstrap）
│   ├── Scripts/                # 脚本目录（13 个 .cs 文件）
│   │   ├── Block/              # 方块模块
│   │   ├── Board/              # 棋盘模块
│   │   ├── Core/               # 核心系统
│   │   ├── Score/              # 计分模块
│   │   ├── UI/                 # UI 模块
│   │   └── Utils/              # 工具类
│   ├── Settings/               # URP 渲染设置
│   └── ...                     # URP 配置文件
├── Packages/                   # Unity 包管理
├── ProjectSettings/            # 项目设置
├── 设计文档/                    # ← 本文档目录
└── README.md
```

> Assets 目录结构的详细规划见 [03-技术架构文档](03-技术架构文档.md)。
