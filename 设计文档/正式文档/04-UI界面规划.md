# 04 - UI 界面规划

> 本文档列出 M3 阶段需要搭建的 UI 界面。  
> 原则：先用已有素材和 placeholder 搭完整流程，后续通过 Prefab 替换正式美术。

## 1. 界面总览

| 界面 | 所属场景 | 说明 |
|------|---------|------|
| Title 启动界面 | `Title` | 游戏启动后的首页，显示模式入口和设置入口 |
| Title 设置弹窗 | `Title` | 音乐、音效、震动、清除用户数据 |
| 传统模式 HUD | 游戏场景 | 当前分、最高分、返回、设置、棋盘和候选区 |
| 游戏内设置弹窗 | 游戏场景 | 音乐、音效、震动、重新开始、返回 Title |
| 游戏结束弹窗 | 游戏场景 | 本局得分、最高分、重新开始、返回 Title |

## 2. Title 启动界面

```text
┌──────────────────────────────┐
│            游戏标题            │
│                              │
│        [ 传统模式 ]           │
│        [ 冒险模式 ]           │ ← 本阶段占位，可置灰
│                              │
│          [ 设置 ]             │
│            v1.0              │
└──────────────────────────────┘
```

**元素清单：**
- 游戏标题或 placeholder Logo。
- 模式按钮列表：由模式配置生成。
- `传统模式`：可点击，进入当前游戏玩法。
- `冒险模式`：只显示按钮，不实现内容，可提示“敬请期待”。
- 设置按钮、版本号。

## 3. Title 设置弹窗

```text
┌──────────────────────┐
│        设置           │
│   音乐      [ 开 ]    │
│   音效      [ 开 ]    │
│   震动      [ 开 ]    │
│                      │
│   [ 清除用户数据 ]    │
│        [ 关闭 ]       │
└──────────────────────┘
```

**元素清单：**
- 音乐开关/音量。
- 音效开关/音量。
- 震动开关。
- 清除用户数据按钮：必须二次确认。
- 关闭按钮。

> Title 设置不放“重新开始”。重新开始属于进入传统模式后的游戏内流程。

## 4. 传统模式 HUD

```text
┌──────────────────────────────┐
│  ← 返回 Title      ⚙ 设置    │
│       最高分 / 当前分          │
│                              │
│          8 × 8 棋盘           │
│                              │
│       3 个候选方块区域         │
└──────────────────────────────┘
```

**元素清单：**
- 返回 Title 按钮。
- 游戏内设置按钮。
- 当前分和最高分显示。
- 棋盘区域和 3 个候选方块。
- 飘字层、音效触发、PC 抖动反馈。

## 5. 游戏内设置弹窗

```text
┌──────────────────────┐
│      传统模式设置      │
│   音乐      [ 开 ]    │
│   音效      [ 开 ]    │
│   震动      [ 开 ]    │
│                      │
│   [ 重新开始 ]        │
│   [ 返回 Title ]      │
│   [ 继续游戏 ]        │
└──────────────────────┘
```

**元素清单：**
- 音乐、音效、震动设置。
- 重新开始当前模式。
- 返回 Title。
- 继续游戏/关闭弹窗。

## 6. 游戏结束弹窗

```text
┌──────────────────────┐
│      游戏结束         │
│      本局得分         │
│      本模式最高分      │
│   [ 重新开始 ]        │
│   [ 返回 Title ]      │
└──────────────────────┘
```

**元素清单：**
- 本局得分。
- 本模式最高分；如果刷新最高分，显示“新纪录”。
- 重新开始。
- 返回 Title。

## 7. UI 技术说明

- 使用 Unity **uGUI**。
- 参考分辨率：**1080 × 1920 竖屏**。
- Canvas Scaler：`Scale With Screen Size`。
- UI 尽量 Prefab 化，位置、图标、音效触发脚本在 Inspector 配置。
- 按钮音效优先挂 `UIAudioTrigger`，不要在按钮点击代码里写死音效。

## 8. 美术素材需求

| 类型 | 当前策略 | 说明 |
|------|----------|------|
| Title 背景 | 先用已有背景或 placeholder | 后续可替换正式标题背景 |
| 模式按钮 | 先用现有主按钮素材 | 传统模式、冒险模式共用样式即可 |
| 设置图标 | 已有 music / sound / vibration | M3 优先接入 |
| 弹窗面板 | 先用简单半透明面板 | 后续替换正式九宫格面板 |
| 音效资源 | 先用 placeholder | 正式音效放到后续 M4/M5 打磨 |

> 详细资源路径见 [08-美术资源替换指南](08-美术资源替换指南.md)。
