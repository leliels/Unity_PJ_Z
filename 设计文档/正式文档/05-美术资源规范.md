# 05 - 美术资源规范

> 本文档面向美术和特效同学，说明所有需要提供的素材、格式要求和命名规范。  
> 请按照本文档的规范来命名和导出资源，方便统一管理。

## 1. 通用规范

### 1.1 图片格式

| 类型 | 格式 | 说明 |
|------|------|------|
| 精灵图（方块、棋盘等） | **PNG** | 透明背景，无压缩 |
| 背景图 | **PNG** 或 **JPG** | 不需要透明的可用 JPG |
| UI 图片 | **PNG** | 九宫格切片的用 PNG |
| 特效贴图 | **PNG** | 透明背景 |

### 1.2 色彩模式

- 使用 **sRGB** 色彩空间导出（Unity 项目虽然是 Linear，但纹理导入时会自动转换）
- 不要使用 CMYK

### 1.3 命名规范

**格式：`类型_名称_状态.png`**

| 类型前缀 | 含义 | 示例 |
|---------|------|------|
| `blk_` | 方块 | `blk_red.png`、`blk_blue.png` |
| `brd_` | 棋盘 | `brd_cell_normal.png`、`brd_cell_highlight.png` |
| `bg_` | 背景 | `bg_title.png`、`bg_game.png` |
| `btn_` | 按钮 | `btn_primary_normal.png`、`btn_primary_pressed.png` |
| `ico_` | 图标 | `ico_star.png`、`ico_lock.png` |
| `pnl_` | 面板 | `pnl_popup_bg.png` |
| `fx_` | 特效 | `fx_clear_particle.png` |

**命名规则：**
- 全部小写
- 用下划线 `_` 分隔
- 状态后缀：`_normal`、`_pressed`、`_disabled`、`_highlighted`
- 不要用中文、空格、特殊字符
- 音效文件也遵循小写英文 + 下划线命名，例如 `snd_block_pick.wav`

## 3. 特效需求列表

> 面向特效同学。

| 特效名称 | 触发时机 | 描述 | 优先级 |
|---------|---------|------|--------|
| 方块消除 | 行/列被消除时 | 方块碎裂/发光消散效果 | 高 |
| 消除颜色覆盖 | 消除前瞬间 | 被消除的整排先变成放置方块的颜色，短暂停留后再执行消除动画（视觉提示） | 高 |
| 连消特效 | 同时消除多行/列 | 比单消更华丽的效果 | 中 |
| 得分飘字 | 获得分数时 | "+10" 数字上飘并消失 | 高 |
| 方块放置 | 方块落到棋盘上 | 轻微的落地弹跳/光效 | 低 |
| 过关庆祝 | 关卡通过时 | 撒花/烟花效果 | 中 |
| 星星点亮 | 结算时显示星级 | 星星依次亮起的动画 | 低 |

### 特效技术说明

- 推荐使用 Unity 的 **Particle System**（粒子系统）
- 也可以使用 **Sprite 序列帧动画**
- 特效 Prefab 统一放在 `Assets/Prefabs/Effects/` 目录
- 粒子贴图放在 `Assets/Art/Effects/` 目录

## 4. 动画需求

| 动画名称 | 对象 | 描述 |
|---------|------|------|
| 方块拖拽 | 候选方块 | 拾起时略微放大，拖拽跟随手指 |
| 方块落入 | 棋盘方块 | 放入棋盘时的轻微弹跳 |
| 行列消除 | 棋盘格子 | 消除前的闪烁/缩放，然后消失 |
| 按钮点击 | UI 按钮 | 按下缩小，松开恢复 |
| 弹窗出现 | UI 面板 | 从小到大弹出 + 轻微弹性 |

> 大部分动画使用 Unity 原生协程或 Animator 实现，美术先提供静态素材即可。  
> 当前确认：MVP 阶段不引入 DOTween，保持零外部依赖。

## 5. 字体需求

| 用途 | 建议 |
|------|------|
| 游戏标题 | 艺术字体（美术手绘或找免费商用字体） |
| 正文/按钮文字 | 思源黑体 / Noto Sans CJK（免费商用） |
| 数字（分数等） | 等宽数字字体，清晰易读 |

> **注意**：字体需要确认是**免费商用**的，避免版权问题。

## 6. 美术风格参考

> **待补充：需要团队讨论确定美术风格。**

参考方向（来自《快乐爱消除》）：
- 霓虹灯/发光方块风格
- 深色背景 + 彩色方块
- 色彩鲜明、对比度高
- 整体风格：休闲、明亮、讨喜

> 建议美术同学先出 2~3 张风格概念图，团队一起选定方向后再批量制作。
