# 01 - 项目总览

## 1. 项目基本信息

| 项目 | 内容 |
|------|------|
| **项目暂定名** | 快乐消消乐（待定） |
| **游戏类型** | 2D 休闲益智 · 方块消除 |
| **参考游戏** | 《快乐爱消除》（App Store / TapTap 可搜到） |
| **项目目标** | 做一个最小化可玩的完整消除游戏，跑通从开发到上线的全流程 |
| **目标平台** | 先做 PC（Standalone），后续考虑移动端 |

## 2. 团队成员与分工

| 角色 | 主要职责 |
|------|---------|
| **技术美术（你）** | 项目统筹、驱动 AI 编写代码、整合资源、构建项目 |
| **美术** | 游戏内所有视觉素材：方块、棋盘、背景、UI 图标等 |
| **策划** | 游戏玩法、关卡设计、用户交互等 |

> 注：程序部分主要由 AI 辅助完成，技术美术负责驱动和验收。

## 3. 技术环境

| 项目 | 版本 / 说明 |
|------|------------|
| **Unity 版本** | Unity 6 (6000.3.8f1) |
| **项目模板** | Universal 2D |
| **渲染管线** | URP 2D (Universal Render Pipeline 17.3.0) |
| **输入系统** | New Input System 1.18.0 |
| **UI 方案** | uGUI (Canvas + EventSystem) |
| **色彩空间** | Linear |
| **默认分辨率** | 1920 × 1080 |
| **脚本语言** | C# |
| **版本控制** | Git |
