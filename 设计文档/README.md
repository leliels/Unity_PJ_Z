# 设计文档索引

> 本目录为项目「快乐消消乐」（暂定名）的文档汇总入口。  
> 正式结论以 `正式文档/` 为准；会议记录和临时讨论不直接作为实现依据。

## 目录结构

| 目录 | 定位 | 说明 |
|------|------|------|
| [`正式文档/`](正式文档/) | 正式设计与技术文档 | 经过整理后的项目规则、架构、计划和规范，作为开发依据 |
| [`操作指南/`](操作指南/) | 操作类说明 | 工具、资源替换、图集生成等具体操作流程 |
| [`会议内容/`](会议内容/) | 原始记录 | 会议、口述、临时想法和待整理内容，不属于正式设计文档 |

## 关于「会议内容」目录

`会议内容/` 下的文件仅作为原始记录和讨论素材：

- 可以记录未整理的想法、口述内容、会议纪要和临时待办。
- 不作为正式开发依据；需要执行的结论应整理到 `正式文档/` 或 `操作指南/`。
- AI 开发时应优先读取 `正式文档/`，只有在用户指定追溯背景或整理需求时再参考 `会议内容/`。
